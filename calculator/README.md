## Calculator function


Currently the calculator function consists of a single main function with input/output and cmath. It supports a light cli that asks for two numbers and a binary operation. 

Supported binary operations include:
 - Square roots(s)
 - Addition
 - Subtraction
 - Multiplication
 - Division(but not by zero, /)
 - Exponentiation(^)

 It asks for the two numbers and the binary operation and outputs to std the result in the form of an equation.
